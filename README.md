#  Student Record System (C Language)

A simple console-based *Student Record System* built using C programming language.  
This project uses *file handling* to store student data permanently in a binary file.

---

##  Features

-  Add new student record
-  Display all student records
-  Data stored permanently using file handling
-  Continuous menu system
-  Simple and fast execution

---

##  Technologies Used

- C Language
- File Handling (fopen, fwrite, fread, fclose)
- Structures (struct)
- Standard I/O Library
